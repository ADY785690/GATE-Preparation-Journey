# Transactions and ACID PYQs

## PYQ 1

Which ACID property ensures that either all operations execute or none?

A) Consistency

B) Isolation

C) Atomicity

D) Durability

Answer: ✅ C) Atomicity

Why not others?

A) Valid state

B) Concurrent execution

D) Permanent storage

---

## PYQ 2

Which property ensures committed data survives system crash?

A) Atomicity

B) Consistency

C) Isolation

D) Durability

Answer: ✅ D) Durability

---

## PYQ 3

Which property prevents interference among concurrent transactions?

A) Atomicity

B) Isolation

C) Consistency

D) Durability

Answer: ✅ B) Isolation
