# Process Scheduling

## FCFS
First Come First Serve

### Advantages
- Simple
- Easy to implement

### Disadvantages
- Convoy effect
- High waiting time

## SJF
Shortest Job First

### Advantages
- Minimum average waiting time

### Disadvantages
- Starvation possible

## Round Robin

### Formula

Waiting Time = Turnaround Time - Burst Time

### Features
- Uses Time Quantum
- Fair CPU allocation

## Priority Scheduling

Highest priority process executes first.
