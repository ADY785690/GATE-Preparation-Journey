# Process Synchronization PYQs

## PYQ 1

Which condition ensures only one process enters the critical section?

A) Progress

B) Bounded Waiting

C) Mutual Exclusion

D) Starvation

Answer: ✅ C) Mutual Exclusion

Why not others?

A) Progress → Selection of next process

B) Bounded Waiting → Prevents infinite waiting

D) Starvation → Problem, not a solution

## PYQ 2

Race Condition occurs when:

A) Only one process accesses data

B) Multiple processes access shared data simultaneously

C) CPU is idle

D) Memory is full

Answer: ✅ B) Multiple processes access shared data simultaneously

## PYQ 3

Peterson's Algorithm is applicable for:

A) One Process

B) Two Processes

C) Three Processes

D) Multiple CPUs

Answer: ✅ B) Two Processes
