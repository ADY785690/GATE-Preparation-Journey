# Process Scheduling PYQ

## Question

Processes:

| Process | Burst Time |
|----------|------------|
| P1 | 24 |
| P2 | 3 |
| P3 | 3 |

Using FCFS Scheduling, find average waiting time.

---

Execution Order

P1 → P2 → P3

Waiting Time

P1 = 0

P2 = 24

P3 = 27

Average Waiting Time

= (0 + 24 + 27)/3

= 17 ms
